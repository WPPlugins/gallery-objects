Gallery Objects is a highly customizable Gallery Plugin for WordPress, incorporating jQuery flash-like gallery scripts from multiple developers.

== Description ==

WordPress Gallery Objects is a highly customizable Gallery Plugin for your
WordPress site.  This plugin was designed to incorporate jQuery flash-like
gallery scripts developed by multiple different developers with varying styles
and configurations.

Don't limit yourself to one gallery style, and don't lock yourself to
flash.  Switch between views/styles with a few clicks without having to
rebuild your album or losing your old album.

The first version of this plugin incorporates the awesome AD Gallery script
developed by Anders Ekdahl over at CoffeeScripter.com

== Installation ==

See the detailed documention over at
<a href="http://galleryobjects.com/support/documentation/">http://galleryobjects.com/support/documentation/</a>

1. Upload all gallery-objects files to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the WordPress media settings
4. Upload images using the WordPress add media function
5. Create an album
6. Add images to the album
7. Create a view
8. Tweak the view settings
9. Insert the view onto a page or post. [galobj viewid=VIEWID]
